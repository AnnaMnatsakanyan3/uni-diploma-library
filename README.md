 University Digital Repository System

A full-stack university library and academic repository platform developed as a diploma project.

## 🔧 Tech Stack
- React.js (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)
- GitHub (Version Control)

## 👥 User Roles
- Student — search & download materials
- Lecturer — upload resources
- Admin — approve and manage uploads

## 📚 Supported Resources
- Books
- Diploma Works
- Theses
- Methodical Guides
- Exam Preparation Materials

## ⚙️ Features
- Authentication & role-based access
- Repository metadata model
- Category & type filtering
- Search system
- Lecturer uploads
- Admin approval workflow
- Download statistics tracking
- PDF validation
- Sidebar navigation UI

## 🛡 Security Measures
- Password hashing (bcrypt)
- File type validation (PDF only)
- Role authorization
- Parameterized SQL queries

## 🎓 Purpose
Designed to model a real university digital repository for academic materials and student diploma works.
